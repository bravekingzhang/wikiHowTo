# wiki_howto_zh

Wiki How to 的中文版本，觉得挺有意思的，希望大家一起把这个项目维护起来

## 介绍


##### 可以扫码这个二维码体验

![添加二维码体验](https://github.com/bravekingzhang/wikiHowTo/blob/master/art/1583063972.png)

目前支持的功能有：

1. 支持显示推荐列表
2. 支持搜索，但是只能搜索到英文的，因为api不支持搜索到中文的
3. 查看详情
4. 详情里面播放视频的功能
5. 显示html文本
6. 支持收藏
7. 支持从收藏移除
8. 对请求做了缓存，先请求缓存，在请求网络。
